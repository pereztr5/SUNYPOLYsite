Home

The Network and Computer Security Club is dedicated to helping members learn more about computer security in a hands-on fashion. We have an open hacker lab in Kunsela C107. It is open to anybody curious about what we do. If there are people there, you are welcome. Any questions can be answered by officers that are in the clubroom. Like anything else, we are always trying to improve our website. Any suggestions or comments can be sent to info@ncsclub.cs.sunyit.edu
