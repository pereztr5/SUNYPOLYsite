Snorby Install

by Tony Perez

Posted on March 31, 2015 at 11:05 PM

Followed this guide to install Snorby but came across the following problems:

  • Wrong Ruby version (2.2)
  • Could not install older Ruby version from source
  • Permissions were not set correctly
  • Problem using Phusion Passenger

Solutions:

  • Installed Ruby 1.9.3
  • Used using rvm
  • Gave the mysql database the correct user and the snorby files in the web directory
  • Unistalled Phusion Passenger
