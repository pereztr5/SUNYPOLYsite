Aggregation

Get service and the last status for that team

Query

db.results.aggregate(
    {"$match": {"type": "Service"}},
    {"$group": 
        {
            "_id": {"type": "$group", "tnumber": "$teamnumber", "tname": "$teamname"},
            "status": {"$last": "$details"}
        }
    },
    {"$group": 
        {
            "_id": "$_id.type",
            "teams": {"$addToSet": {"teamnumber": "$_id.teamnumber", "teamname": "$_id.tname", "status": "$status"}}
        }
    }
)

Group by challenge and get the total amount of challenge and the total points

Query

db.challenges.aggregate(
    {"$group": 
        {
            "_id": "$group",
            "total challenges": {"$sum": 1},
            "total points": {"$sum": "$points"}
        }
    },
    {"$sort": {"_id": 1}}
)
